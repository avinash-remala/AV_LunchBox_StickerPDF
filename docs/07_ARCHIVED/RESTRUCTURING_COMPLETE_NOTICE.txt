MODULE OVERVIEW
================================================================================

av_lunchbox_stickerpdf.core
  - models.py: Order, Summary, BoxType, RiceType classes
  - pdf_generator.py: PDFGenerator class for PDF creation

av_lunchbox_stickerpdf.data
  - sheets_handler.py: GoogleSheetsClient, OrderExtractor
  - image_extractor.py: ImageOCRExtractor for image-based orders

av_lunchbox_stickerpdf.report
  - summary_generator.py: SummaryGenerator, SummaryWriter classes

av_lunchbox_stickerpdf.cli
  - main.py: CLI class and main() entry point

av_lunchbox_stickerpdf.config
  - app_config.py: AppConfig for settings and paths
  - logging_config.py: Logging configuration

av_lunchbox_stickerpdf.utils
  - file_utils.py: File and directory utilities
